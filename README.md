# JSON Navigtion

JSON Navigation is an app to navigate through .json file.
## How to use

- Run "main.py"
```
Enter a path to file:
```
- Enter path and click Enter
- The app will display a structure of file


#### An error can occure if:
- path is incorrect
- file does not exist
